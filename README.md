# Advent of Code 2025

Solutions for [Advent of Code 2025](https://adventofcode.com/2025) written in Go.

## Structure

Each day has its own directory with two parts:

- `day1/p1/` - Part 1 solution
- `day1/p2/` - Part 2 solution

## Running Solutions

To run a specific day's solution:

```bash
cd dayX/pY
go run solution.go
```

## Setup

Initialize the Go module:

```bash
go mod init adventofcode2025
```
